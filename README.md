## Project Description

Sweet Cake Shop is a PHP/MySQL web application designed for browsing cake products, user account creation, and dynamic record management. It features client-side input validation, password hashing, session management, and backend database CRUD operations.

### Objectives

* Build a responsive cake shop user interface.
* Implement a secure user registration and login system.
* Sanitize and validate inputs on the client and server sides.
* Manage persistent application records using PHP and MySQL CRUD operations.
